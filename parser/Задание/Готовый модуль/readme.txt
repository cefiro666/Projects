Приложение parser работает в интерфейсе командной строки. В качестве аргумента
принимается URL-адрес ресурса, с которого необходимо получить данные.

Имеется конфигурационный файл conf.txt, в котором можно настроить максимальную 
ширину выводимой строки в символах, а также добавить слова, которые должны 
исключаться из итогового текстового файла (каждое новое слово добавлять в конец 
файла, через перенос строки). Для примера несколько слов уже вписаны.
